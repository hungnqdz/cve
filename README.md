# How to setup
Install docker and docker compose

Run this command

```sh
git clone https://github.com/hungnqdz/cve.git
cd ./cve
chmod +x make.sh
docker-compose up
./make.sh
```
If any problems happen with wp_internal, try to remove wordpress_data and db_data
